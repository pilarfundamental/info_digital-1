Ejemplo de infografías

Las infografías son representaciones gráficas de información, como lo dice su nombre. Se usan elementos visuales para mostrar la información de forma didáctica y que se entienda de mejor manera. Los recursos gráficos utilizados no son adornos ni decoraciones, sino elementos claves en la información que se entrega. Una buena infografía requiere poco texto que acompañe la gráfica y elementos gráficos que: cuentan con un hilo narrativo, una jerarquía de la información y son suficientes para entender todo lo que se quiere transmitir o exponer. A continuación se muestran diversos ejemplos de infografías para entender de mejor manera sus componentes y la variedad de formas que puede tomar.
